# AI Code Assistant - Python ChatBot

Production-level developer assistant powered by Groq API and Streamlit.

## Features
- Fix Errors, Improve Code, Generate Code, UI Builder, Explain Code, Create Idea
- 3 specialized Groq models for optimal task performance
- Glassmorphism dark UI with smooth animations
- Code conversion (Python ↔ JavaScript) advanced tool

## Setup
1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `GROQ_API_KEY` in `.env` file
4. Run: `streamlit run main.py`
